## 简介

基于 [holiday-cn][holiday-cn] 的中国节假日 API

## 接口

### 获取节假日信息

其中 `date` 为可选参数，格式为 `yyyy-MM-dd`，不传则默认为当天

```http
GET /?date=2024-02-04
```
